## Introduction
[Spring cloud config server](https://docs.spring.io/spring-cloud-config/docs/current/reference/html/#_spring_cloud_config_server) provides a central place for configuration files, which is perfect for a cloud environment.

### Config Repository
The [Config](src/main/resources/config) directory is used for local testing, while a GitHub repository is used in a cloud environment.
